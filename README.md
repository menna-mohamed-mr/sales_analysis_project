# Sales Analysis Project

## Project Overview
This project focuses on analyzing sales data to uncover business insights and support decision-making.

## Tools Used
- Excel
- SQL
- Power BI

## Project Steps
1. Data Cleaning & Validation using Excel
2. Data Exploration & Analysis using SQL
3. Building an Interactive Dashboard using Power BI

## Key Insights
- Discounts higher than 20% can reduce profitability.
- Certain product categories generate higher revenue.
- Sales performance varies across regions and customer segments.
